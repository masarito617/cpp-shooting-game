# 2Dシューティングゲーム
C++で開発したシンプルな2Dシューティングです。
<br>
Buildフォルダに格納してあるEXEファイル、UNIX実行ファイルをそのまま実行して遊べます。
<br>
※ビルドする場合は、SDL2、SDL2_imageをインストール後、パスを通す必要があります。
<br>
　(Macの場合はbrewでインストール後、CMakeLists.txt内の記述を変更すればOK)
<br>
<br>
[開発環境]
<br>
開発PC：Mac
<br>
使用IDE：CLion
<br>
<br>
[操作説明]
<br>
SPACE, ENTER：スタート
<br>
W, A, S, D：上下左右移動
<br>
K：ミサイル発射
<br>
<br>
[操作説明]
<br>
カニを全て倒したらクリア
<br>
カニを逃すかぶつかったらゲームオーバー
